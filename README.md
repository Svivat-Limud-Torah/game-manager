# 🎮 KidScreen — מערכת ניהול זמן מסך לילדים

**אפליקציית Electron שמאפשרת להורים לנהל זמן מסך, שאלות ידע, חנות פרסים, ומשחקים לילדים שלהם — הכל בממשק אחד.**

## ✨ תכונות עיקריות

| תכונה | תיאור |
|-------|-------|
| ⏱️ **ניהול זמן מסך** | מגבלת זמן יומית לכל ילד, חלונות זמן מותרים, מצב חסימה |
| 👨‍👧‍👦 **לוח הורה** | טבלת ילדים, שינוי סיסמאות, מתן מטבעות/זמן, מחיקת חשבונות |
| ❓ **שאלות ידע** | ענה על שאלות והרווח מטבעות — עם מעקב התקדמות |
| 🛍️ **חנות פרסים** | קנה משחקים ואביזרים עם המטבעות שנצברו |
| 🎮 **ספריית משחקים** | הפעל משחקים שנרכשו ישירות מהאפליקציה |
| 🏆 **מערכת הישגים** | XP, רמות, רצפים, ולוח מצטיינים |
| 🔔 **הודעות הורה** | שלח הודעות לילדים שיופיעו במסך הנעילה |
| ☁️ **סנכרון Firebase** | גיבוי ענן, ריבוי מכשירים, סטטוס מחובר בזמן אמת |
| 🔒 **שחזור גישה** | PIN / קוד משפחה / הסרת התקנה מאובטחת |

## 🚀 התקנה

### דרישות מקדימות
- [Node.js](https://nodejs.org/) גרסה 18 ומעלה

### צעדים

```bash
# שכפל את הריפו
git clone https://github.com/<YOUR_USERNAME>/game-manager.git
cd game-manager

# התקן את כל החבילות
npm install

# הפעל
npm start
```

### מצב פיתוח (עם DevTools פתוח)

```bash
npm run dev
```

- **מערכת משתמשים** - התחברות והרשמה עם שמירת נתונים

- **חנות** - קנה משחקים ואביזרים עם המטבעות שצברת
- **המשחקים שלי** - הפעל משחקים שרכשת
- **העולם שלי** - צפה באביזרים שרכשת והפעל את העולם האישי (Unity)
- **אתגרים** - אתגרים מיוחדים לשחקנים מקצועיים עם פרסים
- **סנכרון Firebase** - שאלות ונתונים מסונכרנים לענן
